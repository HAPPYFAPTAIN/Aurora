# 事件卡模板

## 卡片结构

### [event] 事件名称

- **简介**：事件 名称。3-5句事件概要、时间地点、关键参与者和影响。最后写明"上下文出现相关内容时，一定要参考本项详情。"
- **重要度**：important / minor
- **标签**：标签1, 标签2
- **关键词**：事件名, 关键人物, 地点

## 影响与后果
（对剧情、角色、世界观的长期影响）

## 关键细节
（重要的具体细节、对话、物件等）

## 提炼要点
1. 事件要有时序性，标明因果链
2. 区分直接结果和长期影响
3. 保留关键转折点的具体描述
4. 如果是系列事件，标明在系列中的位置
