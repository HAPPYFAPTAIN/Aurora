# 分析卡模板

## 卡片结构

### [analysis] 分析主题

- **简介**：分析 主题。3-5句分析角度、核心结论和适用场景。最后写明"上下文出现相关内容时，一定要参考本项详情。"
- **重要度**：important / minor
- **标签**：标签1, 标签2
- **关键词**：分析主题, 关键概念

## 提炼要点
1. 分析卡是对素材的深度解读，不是简单摘抄
2. 区分文本中的事实描述和主观评价
3. 提取可指导创作的规律和模式
4. 保留原文关键表述作为依据
