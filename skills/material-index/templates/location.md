# 地点卡模板

## 卡片结构

### [location] 地点名称

- **简介**：地点 名称。3-5句地点类型、位置、特征和适用场景。最后写明"上下文出现相关内容时，一定要参考本项详情。"
- **重要度**：important / minor
- **标签**：标签1, 标签2
- **关键词**：地点名, 别名, 关键特征

## 提炼要点
1. 地点卡服务于长期反复出现的场所
2. 重点记录环境特征和特殊规则
3. 标注与主线剧情的关联
4. 保留具有辨识度的细节
