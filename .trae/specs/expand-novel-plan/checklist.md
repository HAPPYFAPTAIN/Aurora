# 扩充验证清单

## 字数验证
- [ ] 全部26章 ≥ 8000字/章

## 内容验证
- [ ] 主线不变更（唐瀚凌视角、战友牺牲、战后访家）
- [ ] 所有人物性格和关系保留
- [ ] 新增内容不产生逻辑矛盾
- [ ] 战斗细节符合军事常识

## 风格验证

- [ ] 无"演习里...现在..."对比（叙事层）
- [ ] 无分析性旁白
- [ ] 细节丰富但不啰嗦
